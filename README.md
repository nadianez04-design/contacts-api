# Contacts API

REST API для управления контактами. Учебный проект на Spring Boot.

## Технологии

- Java 21
- Spring Boot
- Spring Data JPA / Hibernate
- H2 Database
- Maven

## Возможности

- CRUD-операции над контактами
- Хранение данных в базе 
- Валидация входящих данных
- Корректные HTTP-коды ответов

## Эндпоинты

| Метод  | Адрес          | Описание              | Код успеха |
|--------|----------------|-----------------------|------------|
| GET    | /contacts      | Список всех контактов | 200        |
| GET    | /contacts/{id} | Контакт по id         | 200 / 404  |
| POST   | /contacts      | Создать контакт       | 201        |
| PUT    | /contacts/{id} | Обновить контакт      | 200 / 404  |
| DELETE | /contacts/{id} | Удалить контакт       | 200 / 404  |
